Description-
The project contains the implementation of A-Star path planning algorithm for a TurtleBot in a space with obstacles. The output of the planner is visualized using Matplotlib. 

Dependencies-
1. Ubuntu 16.04 or Windows
2. Python 3.7 and above

  
Run Instructions-
python3.7 astar-turtlebot.py

Input the start, goal and clearance.
